# E-Commerce SQL Mini Project

## Overview
This project is a beginner-friendly SQL analytics project built using SQL Server.  
The goal of this project is to practice:

- SQL Joins
- Aggregations
- GROUP BY and HAVING
- Revenue analysis
- Business-oriented SQL queries
- Relational database concepts

The project simulates a simple e-commerce database containing customers, products, and orders.

---

# Database Schema

## Customers Table
Contains customer information.

| Column Name | Data Type |
|---|---|
| customer_id | INT |
| customer_name | VARCHAR |
| city | VARCHAR |
| signup_date | DATE |

---

## Products Table
Contains product details.

| Column Name | Data Type |
|---|---|
| product_id | INT |
| product_name | VARCHAR |
| category | VARCHAR |
| price | DECIMAL |

---

## Orders Table
Contains order transactions.

| Column Name | Data Type |
|---|---|
| order_id | INT |
| customer_id | INT |
| product_id | INT |
| quantity | INT |
| order_date | DATE |

---

# Relationships

- customers.customer_id → orders.customer_id
- products.product_id → orders.product_id
